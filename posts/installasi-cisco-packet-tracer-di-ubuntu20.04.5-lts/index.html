Sekilas Tentang Cisco Packet Tracer

Cisco Packet Tracer adalah aplikasi simulator jaringan milik Cisco yang dapat anda gunakan untuk mengasah ketrampilan anda dalam melakukan konfigurasi perangkat jaringan buatan Cisco.

Installasi Cisco Packet Tracer di Ubuntu 20.04.5 LTS

Packet Tracer yang akan di-install adalah versi 8.2 yang merupakan versi terbaru saat tutorial ini dibuat yakni pada tanggal 6 Maret 2023. Berikut ini step by step instalasi Packet Tracer versi 8.2 di Ubuntu 20.04.5 LTS:

Agar prosses installasi berhasil dan bisa menjalankan Cisco Packet Tracer ada beberapa persyaratan yang harus di penuhi, sepsrti dibawah ini

Cisco Packet Tracer 8.2 (64 bit):

  • Komputer dengan operating system : Microsoft Windows 8.1, 10, 11 (64bit), Ubuntu 20.04 LTS (64bit) or macOS 10.14 atau yang terbaru.
  • amd64(x86-64) CPU
  • 4GB of free RAM
  • 1.4 GB of free disk space

Install Paket Dependensi

Packet Tracer versi 8.2 memerlukan beberapa paket yang perlu kita install (paket depedensi) agar aplikasi Packet Tracer dapat berjalan lancar. Paket-paket tersebut, yaitu: dialog, dxdg-utils, gtk-update-icon-cache, libgl1-mesa-glx, libpulse0, libnss3, libxss1, libasound2, libxslt1.1, libxkbcommon-x11-0, dan libxcb-xinerama0-dev.

Bila paket dependensi di atas belum ter-install, gunakan perintah di bawah ini untuk meng-install-nya:

sudo apt update
sudo apt install dialog xdg-utils gtk-update-icon-cache libgl1-mesa-glx libpulse0 libnss3 libxss1 libasound2 libxslt1.1 libxkbcommon-x11-0 libxcb-xinerama0-dev

Jika muncul pesan error silahkan masukan perintah : 

sudo apt --fix-broken install

Download Cisco PT

Silahkan download Cisco Packet Tracer di situs resminya www.netacad.com biasanya kan diminta untuk login, jika belum memiliki akun silahkan buat akun terlebih dahuli agar bisa mengunduh Cisco Packet Tracer versi terbaru. Pilih “Resources” dan Download Cisco Packet Tracer, lalu sesuaikan dengan sistem operasi komputer anda.

Setelah berhasil mendownload lalu buka terminal lalu ketikan perintah

cd Downloads/
ls CiscoPacketTracer_820_Ubuntu_64bit.deb

Install Cisco PT

Masih pada posisi Downloads, Extrak dan install paket tersebut dengan perintah:

sudo apt update
sudo dpkg -i CiscoPacketTracer_820_Ubuntu_64bit.deb

Untuk menjalankannya ketikan perintah :

packettracer

Terimakasih Semoga bermanfaat, Selamat belajar…
