Cara Install Neofetch di Ubuntu

Pada kesempatan kali ini saya akan share tentang Neofetch dan bagaimana cara install Neofetch di Ubuntu atau keluarga Debian. Secara fungsi tool ini mirip dengan Screenfetch, yakni sama-sama menampilkan informasi sistem di terminal. Tetapi Neofetch lebih populer karena tampilannya yang menarik.

Apa Itu Neofetch?

adalah skrip shell sederhana yang berfungsi untuk memindai informasi sistem yang kita gunakan dan menampilkannya di terminal, lengkap dengan logo distro linux yang ditampilkan dalam format ASCII.

seperti yang kita lihat, pada bagian kiri menampilkan logo distro yang saya gunakan, yakni Ubuntu. Menariknya, logo ASCII tersebut dapat dikustomisasi bahkan diganti dengan foto atau gambar biasa.

Sedangkan di bagian kanan menampilkan informasi sistem dan hardware yang kita gunakan. Beberapa informasi yang ditampilkan antara lain:

Informasi yang bisa ditampilkan neofetch
  • Sistem Operasi
  • Versi kernel
  • Jenis Shell
  • Jumlah paket yang terintsal
  • Desktop Environment
  • System Uptime
  • Resolusi Layar
  • Informasi CPU, GPU, RAM

Selain Linux, Neofetch juga mendukung sistem operasi lain seperti MacOS, Unix, iOS, Android, bahkan Windows. Sehingga menjadikan Neofetch sebagai salah satu tool populer untuk melihat informasi sistem.

Install Neofetch di Ubuntu

Untuk keluarga Debian seperti Ubuntu, Linux Mint, Elementary OS dan sebagainya, caranya sama saja. Jika anda menggunakan versi Ubuntu 16.04 ke atas atau Debian 9 ke atas, Neofetch sudah tersedia pada repository resmi.

Pertama-tama lakukan update repository.

sudo apt update

Kemudian install Neofetch menggunakan perintah berikut.

sudo apt install neofetch

Lalu muncul pertanyaan “Do you want to continue? [Y/n]” tekan tombol Y dan Enter untuk melanjutkan instalasi.

Setelah instalasi selesai, anda dapat menjalankan Neofetch menggunakan perintah berikut:

neofetch

Nah, itu dia tutorial singkat tentang cara menginstall Neofetch di Ubuntu, Debian. yang bisa anda lakukan dengan mudah. Semoga artikel ini bermanfaat Terimakasih.
